1. 스위치 누르면 5초 셈, 두번째 스위치 누른 만큼 LCD에 10X10 의 점이 찍힘
(5초 세는동안 스위치 입력의 수 카운팅)

2. 500ms 간격으로 이미지가 회전하는데, SW0 을 누르면 왼쪽, SW1 을 누르면 오른쪽 방향 정해주는대로 간다. 
양측 버튼을 누르면 멈추고 다시 버튼을 누를때까지 대기한다.

3. 네모가 SW0을 누르면 왼쪽으로 이동하고 SW1을 누르면 우측으로 이동한다. (초급)

4. 네모가 자동차처럼 회전하면서 앞으로 나아가는것을 구현 (스네이크 게임처럼) (고급)

5. 사진복붙문제 (원본 -> 흑백 , 원본 -> 회전)

6. 0~9 폰트 이용하여 B,I,U 문자 만들기
